Carstore.mx design system: premium dark industrial-tech fleet safety company, Spanish-language B2B site

## Brand
- Company: Carstore.mx — Mexican fleet safety technology company
- Tone: Corporate, premium industrial-tech, high-trust, commercially sharp
- Language: All UI copy in Spanish
- Primary CTAs: "Ver catálogo" and "Contactar a ventas"
- Logo: src/assets/carstore-logo.png (inverted white for dark backgrounds)

## Colors (HSL) — Dark premium palette
- Background: 220 10% 6% (deep charcoal-black)
- Foreground: 0 0% 95% (near-white)
- Card: 220 10% 9% (elevated graphite)
- Surface dark: 220 12% 4% (deepest black)
- Accent (safety red): 355 78% 52%
- Border: 220 8% 16% (subtle white/dark)
- Muted foreground: 220 5% 55%

## Typography
- Display font: DM Sans (headings, font-display)
- Body font: Inter (body, font-sans)
- Tracking: tight on headings, wide on labels

## Visual DNA
- Grid overlays (grid-overlay, grid-overlay-fine utility classes)
- Red glow accents (glow-red, glow-red-subtle utilities)
- Backdrop blur on cards (bg-card/50 backdrop-blur-sm)
- Borders: border-white/[0.06] pattern
- Hover accents: border-accent/20

## Structure
- Hybrid corporate + boutique catalog, dark theme throughout
- Homepage: Hero → Solutions → Platform → Industries → Catalog → Services → Why Carstore → CTA → Contact → Footer
- All sections alternate bg-background and bg-surface-dark

## Key tech
- Dashcam + ADAS + DSM, radares punto ciego, cámaras IA, 360°, GPT Monitor platform
- Manufacturing in Shenzhen + CDMX

## Design guardrails
- NO startup SaaS clichés, NO soft consumer UI, NO generic dashboard overlays
- NO cyan, purple, gradient borders, glassmorphism
- Keep spacing generous, hierarchy strong, visual restraint
- Red used sparingly — accent only, never dominant
